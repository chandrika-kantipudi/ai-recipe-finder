# 🍽 AI Recipe Finder

A React application that allows users to search recipes using TheMealDB API.

## Features

- Search recipes
- View recipe details
- Recipe modal
- Responsive design
- Hover animations
- Search using Enter key

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- TheMealDB API
